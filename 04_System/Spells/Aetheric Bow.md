# Aetheric Bow

## Description

Conjures a bow forged entirely from aether, allowing the user to create and fire arrows without requiring physical ammunition.

---

## Scaling Effects

| Tier   | Effects                                                                                                                                                                                                                                                                |
| ------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Iron   | Conjures an aether-forged bow for 2 minutes. The user may generate arrows while the bow is active. Arrow damage scales with Arcane Power and spell quality. Summoning cost: 50 uniums. Arrow cost: 8 uniums per arrow. Cooldown: 3 minutes.                            |
| Silver | Conjures an aether-forged bow for 2 minutes. The user may generate arrows while the bow is active. Arrow damage scales with Arcane Power and spell quality. Summoning cost: 90 uniums. Arrow cost: 15 uniums per arrow. Cooldown decreases according to spell quality. |
